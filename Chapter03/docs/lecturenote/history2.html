关键词：热传导、拉格朗日之争、FFT、认知演进 [cite: 7]

1. 前奏：弦论之争与古典萌芽 [cite: 9]

  • 1747年 达朗贝尔：导出波动方程，开启了“波”的数学化 [cite: 10]。
  • 伯努利 (1753)：基于物理直觉提出“叠加原理”，认为乐音是基音与泛音的组合 [cite: 10]。
  • 欧拉 (Euler)：虽然推导出正交性公式，但由于缺乏收敛性概念，对其普适性持怀疑态度 [cite: 10]。

拉格朗日的“断点”阻力 [cite: 12]

1759年，拉格朗日强烈反对三角级数。他的理由直击要害 [cite: 13]：

“三角级数是连续且可微的，怎么可能组合出具有’尖角’和’断点’的折线函数？” [cite: 14]

—— 这种思维限制了数学界近半个世纪 [cite: 15]。

2. 突破：埃及的热浪与直觉 [cite: 16]

1807年，傅里叶提交了关于热传导的论文。他跳出了“弦振动”的束缚 [cite: 16]：

  • 核心断言：任何周期信号都可以用三角级数表示 [cite: 16]。
  • 15年的学术冷战：因拉普拉斯、拉格朗日等人的质疑，论文直到1822年《热的分析理论》才正式发表 [cite: 16]。
  • 史料补充：傅里叶在拿破仑远征期间担任行政长官，其数学洞察力深受实际物理过程（热分布）的启发 [cite: 17]。

3. 完善：从直觉跨入严密分析 [cite: 18]

  • 狄利克雷 (1829)：给出了著名的“狄利克雷条件”。这是数学史上第一次明确了三角级数表示法的适用边界 [cite: 19]。
  • 从级数到变换：通过将周期趋于无穷大，实现了从周期信号（加权和）到非周期信号（积分）的飞跃 [cite: 19]。
  • 认知结论：傅里叶分析不仅是计算工具，更是一种将“时域”映射到“频域”的哲学观 [cite: 19]。

4. 离散化：高斯的秘密遗稿 [cite: 20]

虽然教材通常从1965年讲起，但史料显示 [cite: 21]：

1805年 高斯 (Gauss) [cite: 22]

为计算小行星轨道，已发明类似FFT的技巧。但他认为这只是“小把戏”，未曾发表 [cite: 22]。

18-19世纪 天体观测 [cite: 23]

天文学数据天生是离散的，这为数值分析和傅里叶方法的结合提供了第二舞台 [cite: 23]。

5. 1965：FFT 与数字文明的开启 [cite: 24]

  • Cooley-Tukey 算法：冷战期间为监测核试验而“再发现”FFT [cite: 25]。
  • 计算量级跳变：将 \(O(N^2)\) 降至 \(O(N \log N)\) [cite: 25]。
  • 应用爆发：雷达、通信（4G/5G）、图像处理（JPEG）、语音识别 [cite: 26]。

Note

如果没有FFT，现代智能手机的信号处理将需要几百倍的功耗 [cite: 27]。
