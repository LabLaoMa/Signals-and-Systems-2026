本节核心：信号还原的质量 (P59-89)

在前面的讨论中，我们假设信号可以展开为无穷级数。现在我们必须讨论现实中的近似问题：

  1. 逐点收敛 (Pointwise)：关注波形在每个时刻 \(t\) 的数值精度。
  2. 均方收敛 (Mean-Square)：关注信号总能量的近似程度。
  3. 完备性 (Completeness)：正交基是否能覆盖整个信号空间？
  4. 吉布斯现象 (Gibbs)：间断点处的工程代价。

1. 逐点收敛与 Dirichlet 条件

逐点收敛的定义

若当 \(N \to \infty\) 时，部分和 \(S_N(t)\) 在每个固定的 \(t\) 处均满足： \[\lim_{N \to \infty} S_N(t) = x(t)\]

狄里赫利 (Dirichlet) 条件 (P62)

虽然数学要求严苛，但物理可实现信号几乎均满足： 1. 一周期内绝对可积：\(\int_T |x(t)| dt < \infty\) 2. 一周期内极值点数目有限。 3. 一周期内间断点数目有限。

间断点处的收敛特性 (P65)

在 \(x(t)\) 的第一类间断点 \(t_0\) 处，傅里叶级数并不收敛于 \(x(t_0)\)。

根据证明，级数收敛于该点左右极限的平均值： \[S_\infty(t_0) = \frac{x(t_0^+) + x(t_0^-)}{2}\]

Note工程启示

对于理想的阶跃信号，傅里叶级数在跳变瞬间给出的结果是“中庸”的。

2. 均方收敛与 \(L_2\) 空间 (P70)

能量视角下的收敛

工科学生更应关注误差信号能量： \[e_N = \int_T |x(t) - S_N(t)|^2 dt\]

均方收敛定理

只要信号 \(x(t)\) 是平方可积的（即信号能量有限，\(x \in L_2\)），则： \[\lim_{N \to \infty} e_N = 0\]

即使在跳变点处“逐点”不相等，但由于该点的宽度为零，其能量误差为零。

完备性与希尔伯特空间 (P75)

完备性 (Completeness)

文档强调：三角函数族 \(\{e^{jn\omega_0t}\}\) 是 \(L_2\) 空间的一组完备标准正交基。

  • 没有任何非零的能量有限信号与所有基函数正交。
  • 这保证了傅里叶级数可以表示空间内任何能量有限的信号。

帕塞瓦尔 (Parseval) 定理 (P78)

均方收敛最重要的物理结论：

\[\frac{1}{T} \int_T |x(t)|^2 dt = \sum_{n=-\infty}^{+\infty} |c_n|^2\]

  • 左边：时域总功率。
  • 右边：各谐波分量功率之和。

结论：时域与频域在能量描述上是完全等价的。

3. 吉布斯现象深度分析 (P82)

现象观察

当用有限项 \(N\) 近似不连续信号时，在跳变点附近会出现振荡，且第一个峰值的过冲量始终保持在跳变幅度的约 8.95%。

数学本质 (P86)

  • 级数在 \(L_2\)（能量）意义下收敛。
  • 级数在 \(L_\infty\)（最大偏差）意义下不收敛。

吉布斯现象的工程对策

为什么会出现？

本质上是频域矩形窗截断带来的卷积效应（Sinc 函数旁瓣导致）。

解决方案

  • 加窗 (Windowing)：利用 Hamming 或 Kaiser 窗平滑截断，牺牲上升沿速度换取振铃消失。
  • 实际意义：在控制系统采样或图像边缘处理中，必须考虑这种过冲对阈值判定的影响。

总结：收敛性的双重境界

特性 逐点收敛 (Pointwise) 均方收敛 (\(L_2\))
判定 Dirichlet 条件 平方可积 (\(x \in L_2\))
间断点表现 收敛至算术平均值 能量误差趋于零
核心误差 吉布斯现象 (9.5%) 截断导致的能量损失
对应定理 收敛定理 帕塞瓦尔定理
